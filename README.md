## 服务环境

### node.js
```
（1）从官网 http://nodejs.cn/ 下载对应版本，安装在电脑上，

	一般情况下，node提供的工具包（npm）会随着node的安装，一起安装在电脑上。

（2）使用命令验证安装是否成功

	node -v(node --Version)

	node -h(help)

	node -p

	npm -v
```

## 使用node.js安装对应框架脚手架环境

### Install
```
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

## 部署一个vue脚手架项目

### Create a project:
```
vue create my-project
# OR
vue ui
```

## 运行项目

### run project:
```
执行命令

npm run dve (旧版本)

npm run serve (新版本)

运行成功之后,终端会给出,当前项目的域名和端口号,可以直接将其运行在浏览器中
```
